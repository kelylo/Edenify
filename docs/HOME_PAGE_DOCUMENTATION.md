## Animations

### Framer Motion Features
- Page load: Staggered section entrance (0.1s delays)
- Progress bar: 1s linear fill animation
- Task items: Staggered entrance with 0.03s delays
- Modal: Spring damping (30) for bounce effect
- Pillar cards: Scale 0.9 → 1.0 animation
- Exit animations: Scale down with opacity fade

### CSS Transitions
- Hover states: 0.2-0.3s smooth color/shadow changes
- Button press: `active:scale-95` for tactile feedback
- Border changes: Smooth 300ms transitions
